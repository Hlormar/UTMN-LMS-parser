# UTMN-LMS-parser
## Usage
Provide 2 arguments through the space when launching:
1. Your corporative mail (@study.utmn.ru part is not necessary)
2. Your password

```bash
python3 parser.py stud322@study.utmn.ru Z1Z2Z3
```
